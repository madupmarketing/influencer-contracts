## 무엇을 하는 도구인가

인플루언서 정보와 계약 조건만 입력하면 **매드업 표준계약서(Word .docx)** 를
규칙에 맞게 자동으로 채워 **발송용 계약서 파일**을 즉시 생성해 주는 도구입니다.

- 기준 서식: 매드업 인플루언서 광고 표준계약서 (`표준계약서_원본.docx`)
- 산출물: 인플루언서별 맞춤 `.docx` 파일 (바로 다운로드 → 발송)
- 개인/사업자 구분, 계약금액 주석, 2차 활용 기간 자동 계산 등
  메리츠 파트너스 계약 실무 규칙이 내장되어 일관된 결과물 생성

---

## 3단계 워크플로우

### ① 정보 입력

- **기본 정보**: 계약자 유형(개인/사업자), 이름·사업자명, 브랜드명, 계약건명, 캠페인명, 콘텐츠 주제
- **채널 정보**: 채널명, 채널 URL, 유튜브 링크
- **계약 조건**: 계약금액, 업로드 예정일, 제작 수량, 2차 활용 라이선스 기간
- **지급 & 계약일**: 세금계산서 발행일, 지급 예정일, 입금 계좌, 계약 서명일
- **수임인 상세 정보**: 개인(성명·생년월일·주소) / 사업자(회사명·등록번호·주소·대표자)

### ② 자동 채움 → 검수

- 입력값이 표준계약서의 정확한 위치에 자동 삽입
  - 수임인 표기: `이름(채널명)` 형식
  - 계약금액 주석: 개인 `*3.3% 원천징수 차감 전` / 사업자 `*VAT 별도`
  - 2차 활용 기간: 시작일·종료일 또는 기간 계산기(시작일 포함 계산)
- **댓글·메모 자동 제거** → 깔끔한 발송용 문서로 정리

### ③ 다운로드 → 발송

- **파일명 자동 규칙**: `이름(채널명)_브랜드명_인플루언서_광고_표준계약서_계약일.docx`
- 생성 즉시 다운로드, **생성 내역 탭**에서 이번 세션 기록 재다운로드 가능
- Word에서 바로 열리는 검증된 .docx (XML 유효성 자동 확인)

## 기술 메모 (내부용)

- 프레임워크: Streamlit / 배포: GitHub + Streamlit Community Cloud (푸시 시 자동 재배포)
- 저장소: https://github.com/madupmarketing/influencer-contracts
- 계약서 생성: `.docx`(ZIP+XML)를 `w14:paraId` 기준으로 필드 치환 (`generate_contract.py`)
- 개인정보 보호: 통장사본·사업자등록증 등 민감 서류 업로드/OCR 기능은 의도적으로 미포함
- 생성 내역은 세션 메모리에만 저장 (서버 디스크 기록 없음)
